# bodas
The asymptotic bode diagram in MATLAB

## Formalization

In order to use bodas, we must first decompose our transfer function as a product of:

G(s) = 1/(s+a)  
G(s) = (s+b)  
G(s) = 1/s  
G(s) = s  
G(s) = K 

Please note that bodas CAN NOT handle complex poles/zeros.
